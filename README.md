这是帮助大学生进行少量数据分析的程序，

'tqdm', 'numpy', 'decimal', 'scipy', 'colorama'
不过没安装的话不用挨个安装检测到库缺失会自动安装（如果没出错的话）

# 如何使用
下载下来运行就行了，我相信引导足够清晰

计算一轮后会重复计算下一轮，蛮方便
