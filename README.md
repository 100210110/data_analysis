# 需要什么库
required_libraries = ['numpy', 'decimal', 'scipy', 'colorama']
上面那行，从程序贴过来的，运行后会自动检查库是否安装，而且如果缺失会自动从清华镜像安装，不必担心库不对
colorama没用上，想给输出变色来着，但反而不好看了，但库装都装了，指不定啥时候加个颜色
